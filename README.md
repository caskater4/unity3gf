# unity3gf
A complete framework for developing games with Unity 3.x.

# Features
* Fast GUI System (<1ms render time)
  * Animated Textures
  * Buttons
  * Labels
  * Sprites
  * Toggle Switches
  * Videos
  * Tween Animation
* Asset/Text Localization
* Persistent Save System
* Smooth Follow Camera
* Achievement System
* Sound / Music System
* Scene Management
* Scene Loader
* Bumper Video Support
* Global Configuration System
